# 🎨 Copywriting Secrets — Squeeze Page (Built with Elementor Pro)

I used an HTML/CSS/JS squeeze page as a design reference, then rebuilt the whole thing inside **Elementor Pro** — block by block, widget by widget.

## 🛠 Built With

- **Elementor Pro** (WordPress page builder)
- Custom CSS via Elementor's built-in CSS editor
- Elementor Form widget for lead capture
- Countdown widget for the timer
- Motion effects for scroll animations

---

## 📐 What I Recreated

| Section | Details |
|---|---|
| ⏱ Countdown bar | Urgency timer at the top |
| 🏆 Hero | Big headline, eyebrow label, italic subtext |
| 📖 Problem block | Left-border quote style, body copy |
| 📋 Secrets list | Numbered gold bullets with descriptions |
| 💬 Testimonials | Quoted cards with attribution |
| 📩 CTA + Form | Email capture with scarcity badge |
| 🔒 Footer | Links + copyright |

---

## 💡 What I Learned

- How to match a coded design inside a visual builder
- Using **Custom CSS** in Elementor to handle details the widgets can't do alone
- Keeping the layout clean and consistent across sections
- Making it look good on both **mobile and desktop**
